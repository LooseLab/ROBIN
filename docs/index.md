# Welcome to ROBIN

ROBIN (Rapid nanopore brain intraoperative classification) is a powerful tool for real-time classification of nanopore sequencing data, specifically designed for brain intraoperative applications.

## Features

- Real-time nanopore data analysis
- Brain tissue classification
- Intraoperative decision support
- High-performance processing
- User-friendly interface
